# FBN Control Centre — Quick Cheat-Sheet

A plain-English guide to the new bits. Keep it open while you test.

---

## 1. Operations → "Day" (your daily worklist)
**What it's for:** run a day from collection to delivery as simple lists, not cards.

- **Collections** tab: everything to collect. Tick a few going the same way → **Group → build manifest** (pick the line-haul truck + trailer size + driver). It emails the receiving depot what's coming.
- **Line-haul** tab: the trucks in transit. When they arrive, hit **Receive** → the loads drop onto Deliveries.
- **Deliveries** tab: cargo at the depot to deliver. Tick the drops → **Assign trip sheet** (truck + driver + odometer). Then **Mark delivered** → **Get POD**.
- You only see your own depot (DBN sees DBN, JHB sees JHB).

## 2. Trip sheets & manifests
- **Trip sheet** = the driver's delivery run. **Print PDF** (has columns for the client to sign + arrival/delivery times) or **Send to driver** (WhatsApps it to their cell).
- **Manifest** = the line-haul truck to the other depot. It auto-emails the receiving branch the full list: client, packages, weight, cubes, trailer size.

## 3. Broking → Load Board (now a list)
**What it's for:** see brokered loads by stage.
- Filter chips across the top: **To collect → Collecting → Loading → On route → Arrived → POD outstanding → POD received → Closed.**
- Each row has the right button: Assign, advance a step, Get POD, or **Close** when the POD's in.

## 4. Split one load across several trucks
**When:** one client order, but 2–4 subbies carry it.
- Open the load → **🚚 Split trucks** → add a truck per subbie, type each one's rate, tick "email loadcons now".
- Each subbie gets ONE loadcon (with both trucks if they have two). The client is invoiced **once** and gets **one update listing every vehicle**.

## 5. Offer a load to carriers (NEW marketing)
**What it's for:** push a load to the carriers who can do it.
- On an unassigned load → **📣 Offer to carriers**.
- It shows the carriers who already run that **lane + truck type** (from their history). Tick them → **Send offer**.
- They reply with their rate by email. (No client name/your rate ever goes out.)

## 6. Your CRM (Clients + Transporters)
- **Clients** are grouped by category: **Agent / Consolidator / Broker / Manufacturer / Carrier.** Use the chips to filter.
- **🎯 Leads** button = everyone who's ever asked you for a rate — your sales hit-list (company, name, title, email).
- Wrong list? **→ Transporter** moves a client to carriers; **→ Client** moves a carrier back.
- Names show in CAPS, cells in +27, automatically. Add or edit anyone with **Add Client** / **Edit**.
- **Quotes page:** open quotes now show a green **✓ Info req <date>** when you've chased the client for more info.
